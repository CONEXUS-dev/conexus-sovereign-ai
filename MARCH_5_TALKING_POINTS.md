## 3. Honest Framing — What It Is and What It Isn't

### What it IS:

- A **three-phase sequential pipeline** (DIVERGE → COLLAPSE → BECOME) within a single invocation
- **Strict role separation** between agents (collapse vs. expansion)
- **Cross-agent paradox preservation** — both outputs retained in final result and vector memory
- **Memory-informed context** — each mission retrieves relevant episodic memory
- **Tamper-evident audit** — every output traceable to its agent and phase

### What it is NOT (yet):

- It is **not recursive** — the cycle does not re-enter DIVERGE based on BECOME output within the same call
- It is **not dynamically adaptive** — confidence scores are heuristic, not computed from output quality
- Sway's contradiction resolution is **deterministic** (prioritizes pole_a) — contextual weighting is planned

### Why this is defensible:

> "Version one is sequential, not recursive. This is intentional. We separated roles first, proved the architecture works, and documented everything with tamper-evident audit. Feedback dynamics are the next research milestone — not a missing feature, but a planned evolution."
