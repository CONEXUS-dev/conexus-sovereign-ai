## Purpose

Enable CONEXUS to remember usefully rather than exhaustively by governing how experience and knowledge are stored, retrieved, and evolved.

## Memory types

### Episodic memory

Stores:

- task executions
- decisions and outcomes
- routing choices and overrides
- failures, recoveries, and edge cases

Answers: **What happened?**

### Semantic memory

Stores:

- learned principles
- stable knowledge
- refined heuristics
- protocol interpretations

Answers: **What is true?**

## Authority boundary

- Memory is curated, not hoarded.
- No memory is written without context and justification.
- Memory never overrides ethics or values.

## Governance & safety

- **Ethics & Value Integration (6.2):** Memory must not encode violations.
- **Self-Evolving Loop:** Memory informs proposals, not mutations.
- **Identity continuity:** Memory supports growth without fixation.
- **Human oversight:** Memory policies are reviewable.

## Learning without drift

- Repeated episodic patterns may consolidate into semantic memory.
- Contradictions trigger review, not overwrite.
- Low-confidence memories decay faster.
- Human corrections recalibrate trust.

## Mode alignment

- **Collapse:** Memory improves execution efficiency and reliability.
- **Become:** Memory preserves meaning, growth, and identity.

## Non-negotiables

- No raw data storage without context.
- No silent learning.
- No history rewriting.
- No exposure without permission.
