Core Algorithm Logic (Restatement)

Standard BLS Approach:

Rank all transit candidates by detection power (BLS statistic)

Report top N (typically 3-5)

Result: Maximizes SNR, misses unusual morphologies

FE Approach:

1. Preprocessing (BLS): Generate 500 candidate signals, extract features (period, depth, BLS power)
2. Multi-objective fitness: Compute two orthogonal scores

Primary: Coherence (f₁) = normalized BLS power [0-1] (signal validity)

Secondary: Anomaly (f₂) = structural deviation from typical profile (how unusual)

3. Strategic elimination: Each generation

Sort population by f₁ (coherence)

Eliminate lowest-coherence 70% (forget_rate = 0.3)

BUT retain 12/50 highest-anomaly candidates even if low-coherence (paradox buffer)

4. Selection mechanism: Multi-generational feedback amplifies paradoxes (high f₁AND high f₂)

5. Output: Final population of 50 candidates ranked by paradox score P(c)
